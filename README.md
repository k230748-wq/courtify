# Courtify Web (Demo) 🏟️

Courtify Web is a **frontend-only demo website** built to present the vision, features, and user experience of **Courtify**, an upcoming court booking platform.  
The primary goal of this web version is to **showcase the product concept** and **collect early user interest via a waitlist**, while the **main focus of development is the Courtify mobile app**.

All features displayed on the website use **dummy data** and are intended for demonstration purposes only.

---

## 🎯 Purpose of the Web Demo

- Showcase the **Courtify product idea and UI/UX**
- Present core features visually using mocked data
- Collect early adopters through a **waitlist signup**
- Act as a **landing page + product preview** for the upcoming mobile app

---

## ✨ Features (Demo Only)

- 📝 Waitlist signup for early access users  
- 🏸 Court listings and details UI (mocked data)  
- 📅 Booking flow screens (UI only)  
- 👤 User profile and dashboard (UI only)  
- ⭐ Reviews and ratings (dummy data)  
- 📱 Fully responsive design  
- 🎨 Clean, modern UI components  

---

## 📱 Mobile App First Approach

**Important:**  
The Courtify Web platform is **not the final product**.

The core Courtify experience will be delivered through a **dedicated mobile application**, where real bookings, payments, and user interactions will be implemented.

This web demo serves as:
- A concept showcase
- A marketing and waitlist platform
- A UI/UX reference for the mobile app

---

## 🛠️ Tech Stack

- **Frontend:** React  
- **Styling:** Tailwind CSS  
- **State Management:** Local state / props
